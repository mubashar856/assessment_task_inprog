## Configuration & Execution

- Create Empty database in mysql and add database name in .env file along with hostname, port, username and password
- run composer install command
- run php artisan migrate
- run php artisan db:seed
- run npm install command
- run npm run prod command
- open url < domain  >/stats 
- then on new tab open < domain >/simulate

(First link will open page where data will be broadcasted)
(Second url will simulate the data)
(So after running second url go back to first page data will appear there)
